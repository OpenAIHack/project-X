# Project X - Closer

Closer is a 2-hour hackathon MVP for an AI real-estate lead-to-offer workflow.

It turns a messy buyer lead into:

1. A CRM lead record
2. A structured buyer profile
3. Ranked seeded listing matches
4. An offer-ready packet for licensed agent review

See [PRD_Closer_RealEstateAgent.md](PRD_Closer_RealEstateAgent.md) for the current PRD.
